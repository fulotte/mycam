# MyCam v1.0.0 发布

## 主要功能

- 实时视频预览（局域网）
- 智能运动检测自动拍照
- 云端存储（阿里云 OSS）
- 多端通知（飞书/钉钉）
- 响应式 Web 界面
- 设备管理和历史照片

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/fulotte/mycam.git
cd mycam

# 查看部署指南
# docs/DEPLOYMENT.md
```

## 技术栈

- 固件: PlatformIO + ESP32-Arduino
- 云端: 阿里云函数计算
- 前端: Vue 3 + Element Plus
